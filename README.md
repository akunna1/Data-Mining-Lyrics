This project mines and analyzes song lyrics from some of the most played songs on Spotify to visualize their word content. It cleans the lyrics, removes common stop words and auxiliary verbs, calculates word frequencies, and visualizes the top words using pie charts.

---

## About

* Text cleaning using custom stop words and punctuation removal
* Tokenization and frequency distribution analysis with **NLTK**
* Relative word frequency calculation (as a percentage of total words)
* Visualization of top words via pie charts using **Matplotlib**
* Supports analyzing multiple lyrics files in batch

---

## Features

* Cleans and preprocesses raw lyrics data
* Adds custom stop words for auxiliary verbs and common slang
* Calculates both absolute and relative word frequencies
* Displays number of unique words per song
* Visualizes the top 15 most frequent words per song as pie charts

---

## Technologies & Libraries

* Python
* NLTK (Natural Language Toolkit)
* Pandas
* Matplotlib
* Regular Expressions (re)

---

## Usage

1. Prepare text files containing song lyrics.
2. Add the file paths to the `inputs` list in the script.
3. Run the script to see the word frequency analysis and pie charts.
